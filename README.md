# ⚙️ Como Executar o Projeto

## 1️⃣ Clone o repositório

```bash
git clone https://github.com/041067/Clone-da-tela-de-login-do-instagram-com-Tailwind-CSS.git
```

## 2️⃣ Acesse a pasta do projeto

```bash
cd Clone-da-tela-de-login-do-instagram-com-Tailwind-CSS
```

## 3️⃣ Instale as dependências

```bash
npm install
```

## 4️⃣ Execute o Tailwind CSS

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

## 5️⃣ Abra o projeto

Abra o arquivo:

```bash
src/index.html
```
